# GPU-Optimized Dependencies for Windows with CUDA Support  

This file contains the necessary dependencies for a GPU-optimized setup for the Videotrans project on Windows.

## Dependencies
- torch==1.9.0+cu102  # CUDA 10.2
- torchvision==0.10.0+cu102  # CUDA 10.2
- torchaudio==0.9.0  # For audio support
- tensorflow-gpu==2.5.0  # GPU-optimized TensorFlow
- numpy  # Required for numerical operations
- scipy  # Scientific computing

# Installation
Run the following command to install the pip packages:

```bash
pip install -r requirements-win-gpu.txt
```
